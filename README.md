## Installation
Add this line to your application's Gemfile:

```ruby
gem 'workforce'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install workforce
```

## Contributing
add `.env` to root of repo with `POSTGRES_USER` and `POSTGRES_PASSWORD` defined
`bundle install`

Run Specs with `bundle exec rspec spec`
